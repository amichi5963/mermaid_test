# 課題項目
## 流れ図

- 5ノード以上

## 解答
```mermaid
flowchart LR;
  start([開始]) --> A[階層を生成]
subgraph 階層開始
  A --> B{"現在の階層が1"}
  B -->|true| C["あらたな冒険者がまた塔を登る……"]
  B -->|false| D{"現在の階層を3で割ったあまりは1？"}
  D -->|true| E["難易度増加表示"]
  D -->|false| F["階数のみの表示"]
  C -->join{}
  E -->join
  F -->join
end
  join --> G["移動する方向を入力してください(テンキー):"]
subgraph 階層攻略
  G --> H{"2か4か6か8が入力されたか？"}
  H -->|true| I["プレイヤーの移動先を決定"]
  H -->|false| J{"5が入力されたか？"}
  J -->|true| K["巻物の使用"]
end
```
